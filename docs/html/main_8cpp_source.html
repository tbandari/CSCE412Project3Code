13 * Welcome to the documentation for the Load Balancer Simulation project.
14 * This system is designed to simulate how network traffic is distributed
15 * across a dynamic pool of web servers.
16 *
17 * @section desc_sec Project Description
18 * The simulation handles thousands of clock cycles, generating incoming
19 * requests and routing them based on job type ('S' for Streaming, 'P' for Processing).
20 * It features auto-scaling logic to add or remove servers based on queue capacity,
21 * as well as a basic IP firewall.
22 *
23 * @section notes_sec Notes & Comments
24 * - **Scaling:** Managed by `minQueueFactor` and `maxQueueFactor`.
25 * - **Metrics:** A final snapshot is generated in user specified log file upon completion.
26 * - **Author:** Tanish Bandari
