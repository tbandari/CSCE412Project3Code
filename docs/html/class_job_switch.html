void JobSwitch::routeRequest ( const Request & r)

Evaluates the request's job type and routes it to the correct load balancer.

Evaluates the job type of an incoming request and routes it to the correct load balancer.

Parameters
rThe request to be routed.
  • If the request's job type is 'S', it routes to the streaming balancer. Otherwise, it routes to the processing balancer. It also increments the respective job counters on the chosen load balancer.
  • Parameters
    rThe incoming Request object to be evaluated and routed.
