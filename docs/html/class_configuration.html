Loads and parses key-value pairs from the specified file path.

Parameters

Returns
true if the file was loaded and parsed successfully, false otherwise.
  • Reads the file line by line, ignores comments starting with '#', and extracts key-value pairs separated by '='. Populates the internal map with these pairs.
  • Parameters
